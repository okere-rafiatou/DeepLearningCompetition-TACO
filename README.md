# DeepLearningCompetition-TACO

## Project Overview

This project was developed as part of a Deep Learning challenge focused on object detection.

The goal is to build a model capable of detecting and classifying waste objects in real-world environments using the TACO (Trash Annotations in Context) dataset.

Object detection is a key task in computer vision and plays an important role in environmental monitoring and smart waste management systems.

---

## Challenge Objective

- Train an object detection model
- Detect multiple waste categories
- Improve model accuracy and generalization
- Evaluate performance using standard metrics



##  Technologies Used

- Python
- PyTorch / TensorFlow (update depending on what you use)
- OpenCV
- Jupyter Notebook

---

##  Dataset

The dataset is not included in this repository due to its large size.

You can download it from Google Drive:

https://drive.google.com/drive/folders/1mv3Xs4O6ljAZIZ7aYzl_wPcnqzw17Soe?usp=sharing

After downloading, place the `data/` folder in the root directory of the project:

DeepLearningCompetition-TACO/
│
├── data/
├── detector/
├── notebooks/
└── ...

Make sure the folder structure remains unchanged.



##  How to Run

1. Install dependencies
2. Download and place the dataset in the `data/` folder
3. Run the training script or notebook
